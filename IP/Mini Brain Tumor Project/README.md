# Brain Tumor Detection using MRI Image Processing

## Description
This project detects suspicious tumor regions from MRI brain images using classical image processing techniques.

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib

## Features
- MRI preprocessing
- CLAHE enhancement
- Otsu thresholding
- Morphological processing
- Tumor localization
- Tumor presence analysis

## Dataset
Kaggle Brain MRI Dataset

## Run Project

```bash
py tumor_detection.py
